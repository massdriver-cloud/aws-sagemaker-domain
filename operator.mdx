# aws-sagemaker-domain
Amazon SageMaker Domain and User Profile is a fully managed service from AWS that helps data scientists and developers to prepare, build, train, and deploy high-quality machine learning (ML) models quickly by bringing together a broad set of capabilities purpose-built for ML.

## Use Cases
### Machine Learning Development and Deployment
SageMaker provides a complete set of features to take your ML models from idea to deployment, including data labeling, data preparation, algorithm selection, training, tuning, deployment, and management.

### Data Wrangling
With SageMaker, you can clean, transform, and visualize your data with an easy-to-use visual interface.

### Experiment Management
SageMaker provides tools to track, compare, and evaluate ML experiments and model versions.

### Managed Jupyter Notebooks
SageMaker provides fully managed Jupyter notebooks that require no setup, so you can start analyzing your data quickly.

## Design
This bundle is designed around the speicific use-case of creating a SageMaker model and for storing the model's assets in an S3 bucket.

## Decommissioning
Before decomissioning this bundle, you must delete SageMaker apps associated with the user profile. SageMaker Studio creates a default app for each user profile.
